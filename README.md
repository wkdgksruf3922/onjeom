# 🍞 온점 (ONJUM)
> **"빵이 구워지는 시간을 기다리듯, 우리 삶에도 숨 고를 시간이 필요하기에"**
> 시니어 파티시에의 손길로 당신의 하루에 따뜻한 쉼표를 찍어주는 프리미엄 디저트 브랜드입니다.

---

## 🌟 프로젝트 개요 (Overview)
은퇴 후 제2의 인생을 시작하는 시니어 파티시에들이 주체가 되어, 오랜 세월 다져진 손맛과 정성을 담아 빵을 굽는 사회적 기업 브랜드 웹사이트입니다.

- **사회적 미션:** 시니어의 '느림'을 '깊이'로 재정의하고 지속 가능한 일자리를 제공합니다.
- **브랜드 정체성:** 맛을 통한 사회 공헌 및 세대 간의 정서적 공감 형성.

## 🎨 디자인 컨셉 (Design Key Visual)
- **따뜻한 아카이브:** 인생 이야기가 담긴 사진과 손글씨를 활용한 포근하고 아기자기한 경험.
- **인터랙티브 스토리텔링:** 반죽이 부풀어 오르는 '호흡'의 시간을 시각적으로 구현하여 브랜드의 속도감을 체험.
- **핵심 에셋:** <img src="./assets/logo.svg" > (온점 브랜드 로고)
